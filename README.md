# Hoarding Task Force of Washtenaw (htf.mipath.org)
Welcome to the official repository for the Hoarding Task Force of Washtenaw. This repository hosts the source code and static files for the main site, htf.mipath.org, served via GitHub Pages.

## About Hoarding Task Force
The Hoarding Task Force of Washtenaw is a branch of Pathlight Community Services that provides clutter assistance, workshops, and behavior assessments for hoarding.

### Our website serves as the primary hub for:

- Providing contact information
- Volunteer Opportunities
- Describing programs and services
- Tracking related upcoming events
- Providing a definition for hoarding
